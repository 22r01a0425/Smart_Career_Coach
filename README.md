# Smart Career Coach 🎯

Smart Career Coach is a data science-powered web app that helps users match their resumes to suitable job roles based on skills, using AI and Machine Learning techniques.

## 🔍 Features
- 📄 Resume parsing to extract name, email, and skills
- 🔎 Job role matching based on a custom dataset
- 🤖 ML-based job recommendations (optional add-on)
- ✨ Generative AI integration to enhance resumes (coming soon)
- 🖥️ Streamlit-powered web interface

## 📁 Project Structure

smart-career-coach/
├── src/
│ └── app.py
├── data/

## 🚀 How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run src/app.py
